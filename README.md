# OpenGL 3D Zombie Survival

A 3rd-person wave-based zombie survival shooter built from scratch in C++ using OpenGL. Defend yourself against hordes of zombies and mutants across multiple waves using smooth skeletal animations and a robust custom rendering pipeline.

## Features
* **Skeletal Animation:** Smooth blending between idle, walking, sprinting, and attack animations for both the player and enemies.
* **Wave-Based Spawning:** Progressively difficult waves featuring standard Zombies and high-health Mutants.
* **3rd-Person Camera:** Mouse and keyboard-driven camera that tightly follows the player with dynamic zooming.
* **Custom UI & Rendering:** In-world enemy health bars, screen-space player UI, score tracking, and glowing tracer bullets without depth obscurance.
